## **Shan Font**
ၾွၼ်ႉတႆးတႃႇဝႅပ်ႉ ၊ Shan font for website

မႂ်ႇသုင်ၶႃႈ Shan font ပဵၼ်ၾွၼ်ႉဢၼ်ၶူင်သၢင်ႈၶိုၼ်ႈပုၼ်ႈတႃႇ သႂ်ႇၼိူဝ်ဝႅပ်ႉၾႃႇသႃႇတႆးၵမ်းသိုဝ်ႈ။ ၾွၼ်ႉၼႆႉဢၢင်ႈဢိင်ဢဝ်ၾွၼ်ႉ Panglong Unicode သေ ၶူင်သၢင်ႈၶိုၼ်ႈမႃး ပုၼ်ႈတႃႇႁႂ်ႈၵၢၼ်ႁဵတ်းႁႂ်ႈၵၢၼ် embed ၾွၼ်ႉတႆးၼၼ်ႉ ႁဵတ်းလႆႈငၢႆႈ၊ တူၺ်းလႆႈၼႂ်း browser ၸဵမ်ၾူၼ်းလႄႈၶွမ်း၊ ဢိၵ်ႇတင်းႁဵတ်းႁႂ်ႈၵၢၼ်လူတ်ႊၾွၼ်ႉၼၼ်ႉဝႆး ၼႆယဝ်ႉ။

## ၵၢၼ်ၸႂ်ႉၾွၼ်ႉၼႂ်း HTML/CSS
Copy လႄႈသႂ်ႇ code ပႃႈတႂ်ႈၼႆႉ

```css
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/ShanFont/ShanFont@main/shan.css">
```

ၵႂႃႇသႂ်ႇၼႂ်းၵႄႈ

```css
<head> </head>
```

## ၵၢၼ်ၸႂ်ႉၾွၼ်ႉၼႂ်း WordPress

ၵႂႃႇတီႈ **Apperance > Cutomize > Additional CSS**

Copy လႄႈသႂ်ႇ code ပႃႈတႂ်ႈၼႆႉ
```css
@import url('https://cdn.jsdelivr.net/gh/ShanFont/ShanFont@main/shan.css');

/* Apply Shan Font Globally */
body, html, p, span, div, a, li, ul, ol, h1, h2, h3, h4, h5, h6, blockquote, strong, em, b, i, button, input, textarea, select, label, table, th, td, nav, footer, header, section, article, aside, form, address, figcaption, fieldset, legend, details, summary {
  font-family: 'Shan', sans-serif !important;
  line-height: 1.6em;
  letter-spacing: 0 !important;
  -webkit-font-smoothing: antialiased;
  -moz-osx-font-smoothing: grayscale; 
}
```
